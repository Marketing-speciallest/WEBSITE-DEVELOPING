الرفع والتسليم

Deployment — خطوات الرفع على السيرفر

خطوات الـ Deployment
1
إنشاء قاعدة البيانات

cPanel → MySQL Databases → إنشاء DB + User + رفع SQL dump

2
رفع الملفات

FileZilla / cPanel File Manager → public_html/ + ضبط config.php

3
ربط النطاق + SSL

إضافة A Record → Let's Encrypt Certificate → Force HTTPS في .htaccess

4
إعداد البريد الإلكتروني

SMTP عبر cPanel + PHPMailer → إشعارات Contact Form لبريد العميل

5
اختبار نهائي + تسليم

PageSpeed Test + Mobile Test + Form Test + تسليم بيانات دخول لوحة التحكم

# .htaccess — الإعداد الكامل
RewriteEngine On

# Force HTTPS
RewriteCond %{HTTPS} off
RewriteRule ^ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

# Protect admin from direct access
RewriteRule ^admin/$ admin/login.php [L]

# Block directory listing
Options -Indexes

# Gzip Compression
<IfModule mod_deflate.c>
 AddOutputFilterByType DEFLATE text/css text/javascript
</IfModule>

# Browser Caching
ExpiresByType image/webp "access plus 1 year"
